### Solar Ops System

**Role:** Full Stack / Frontend Software Developer </br>
**Stack:** React, Node.js (Express), MySQL/PostgreSQL

📌 Project Description

Contributed to the development of a Solar Plant Monitoring & Operations platform, designed to track, analyze, and manage solar energy generation and system performance in real time.

The system enables operators to monitor plant efficiency, visualize energy production trends, and respond to system alerts through a centralized dashboard.

Implemented core features such as KPI dashboards, energy analytics, alert systems, and device monitoring interfaces, with a focus on performance optimization and responsive UI design.

Note: This repository contains a demo version with simulated/dummy data. It is intended to showcase development work and does not represent a live production system.

This guide will help you set up, run, and access the Solar Ops system locally.
